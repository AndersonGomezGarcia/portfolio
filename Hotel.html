Hotel el descanso (Sistema gestion Hotelera)

Sistema de Gestión Hotelera (SGH) El Sistema de Gestión Hotelera (SGH) es una solución diseñada para la administración eficiente de reservas, ocupación y servicios en el hotel El Descanso. Implementa una base de datos optimizada, asegurando integridad y normalización hasta la 3FN. Características principales:

  • ✅ Registro y gestión de habitaciones, clientes y reservas.
  • ✅ Administración de servicios adicionales (restaurante, bar, lavandería, etc.).
  • ✅ Perfiles de usuario con distintos niveles de acceso (gerente, recepcionista, cajero, administrador).
  • ✅ Generación de facturas y estadísticas sobre ocupación y cancelaciones.

Tecnologías utilizadas:

  • Modelado de datos: Diagrama Entidad-Relación (Chen)
  • Bases de datos relacionales: Implementación con dependencias funcionales y normalización.
  • Herramientas: Draw.io para modelado, SQL para gestión de datos.
